# 🍽️ Restaurant Recommendation System

A content-based restaurant recommendation system built using 
TF-IDF Vectorization and Cosine Similarity ML on the Zomato dataset.

## 🤖 How It Works
1. TF-IDF converts restaurant features into numerical vectors
2. Cosine Similarity compares all 9,542 restaurants (91M+ comparisons!)
3. Multi-factor scoring ranks by similarity + rating + popularity
4. Results filtered by user preferences instantly

## 📊 Results
- 9,542 restaurants analyzed
- 286 unique TF-IDF features
- 63 recommendation filter combinations
- Fully interactive dashboard
